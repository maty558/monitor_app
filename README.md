# Web Monitor

Flutter appka na sledovanie webstránok. Kontroluje prítomnosť kľúčových slov a cien na zadaných URL adresách.

## Funkcie

- Prihlásenie/registrácia cez email
- Pridávanie monitorov (URL + kľúčové slová + voliteľný cenový rozsah)
- Automatická kontrola každých 10 sekúnd
- Zvukový alarm pri nenájdení textu
- História kontrol
- Pozastavenie/zapnutie monitorov

## Požiadavky

- Flutter SDK
- Backend server bežiaci na `http://192.168.100.2:3000`

## Spustenie

```bash
flutter pub get
flutter run
```
